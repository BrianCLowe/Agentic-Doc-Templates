# Upstream files from Agentic Doc Templates

When the whole template repository was copied into a project, an agent following [`BOOTSTRAP.md`](../BOOTSTRAP.md) moves the repo root `README.md` and `LICENSE.md` here so your project root stays yours.

If this folder is empty, you likely copied only `docs/help/`, `docs/agent/`, and `docs/templates/` — that is the recommended approach. License text: [GitHub LICENSE.md](https://github.com/BrianCLowe/Agentic-Doc-Templates/blob/main/LICENSE.md).
